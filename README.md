# Weather Dashboard Application

This is a simple Spring Boot application that displays the weather forecast for Hà Nội, Vietnam.

## Prerequisites

Before you begin, ensure you have the following installed:
- Java Development Kit (JDK) 17
- Apache Maven

## How to Run

1. **Clone the repository or open the project in your IDE.**

2. **Build the project using Maven.**
   Open a terminal or command prompt, navigate to the project's root directory (`employee-management-system/backend/Task_3`), and run the following command:
   ```bash
   mvn clean install
   ```

3. **Run the application.**
   After a successful build, you can run the application with the following command:
   ```bash
   mvn spring-boot:run
   ```
   Alternatively, you can run the main class `WeatherDashboardApplication` from your IDE.

4. **Access the application.**
   Once the application is running, open your web browser and go to the following URL:
   [http://localhost:8080/weather](http://localhost:8080/weather)
